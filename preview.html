10. Verification Checklist

Toggle the theme several times and verify each item visually.

  • Brand identity konsisten — emerald terlihat sebagai "Tempa green" di kedua mode (bukan dua warna berbeda).
  • Kontras text — body text terbaca nyaman di kedua mode tanpa "menyilaukan" atau "redup".
  • Hierarchy — primary text jelas lebih dominan dari secondary dan muted text.
  • Card depth — di light mode, card terangkat dari background. Di dark mode, card terdefinisi via border.
  • Button hierarchy — primary paling menonjol, secondary jelas tapi tidak compete, ghost paling subtle.
  • Status colors — Tinggi/Sedang/Rendah readable dan visually differentiable di kedua mode.
  • Brand accent border — kelihatan di kedua mode (tidak terlalu pucat di light, tidak terlalu menyolok di dark).
  • Hover state — coba hover card di light mode → shadow harus naik halus, tidak agresif.
  • Focus state — tab/click button → focus ring emerald terlihat di kedua mode.
  • Toggle transition — perpindahan light↔dark halus (200ms ease), bukan jarring snap.
