# Asahlagi — Backend

FastAPI + Python 3.11+. Quiz generation, evaluation, and analysis service for the Asahlagi capstone product.

## Prerequisites

- Python 3.11+
- pip 23+
- (recommended) virtualenv or venv

## Setup

```bash
cd backend
python -m venv .venv
source .venv/bin/activate    # Linux/macOS
# .venv\Scripts\activate     # Windows

pip install -r requirements.txt
cp .env.example .env
```

## Run dev server

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

Backend starts at http://localhost:8000.
- Health check: http://localhost:8000/health
- OpenAPI docs (auto-generated by FastAPI): http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc

## Run tests

```bash
pytest                       # all tests
pytest tests/test_health.py  # one file
pytest -v                    # verbose
pytest -k "submit"           # filter by name
```

## When you implement your module

- Keep services pure where possible (no I/O, no globals)
- Raise `ApiException` (not `HTTPException`, not generic `Exception`) for any user-facing error
- Use error code constants from `app/utils/errors.py` instead of string literals
- All user-facing copy comes from `BRAND.md` §7 — don't invent new strings inline
- Add tests in `tests/test_<your_module>.py`
